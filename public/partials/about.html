About The Project


This ports management tool is built mainly upon Play with scala, angularJs written in both javascript and coffeecript.
All the data info are scrapyed from UNECE by a python tool called Scrapy.
To start off, please pick a country code that the port you're looking for is located in. Then you can do loads interesting things from there (At least I believe so...)
I really hope you can enjoy the time spent here, if not, then please enjoy your tea and sandwich...




Commodity Vectors is in charge of predicting to where large cargo ships are headed. To fulfil this mission the company must keep up to date records about all the ports around the world and the vessels it tracks.
